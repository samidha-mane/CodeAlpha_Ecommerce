# CodeAlpha E-Commerce Store

A mini e-commerce web application built as part of my **CodeAlpha internship project**.  
This project demonstrates basic frontend functionality like product listing, shopping cart, and order processing.

## Features
- Product listings
- Add to cart functionality
- Shopping cart with total price
- Basic order placement

## Tech Stack
- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** To be added (Express.js / Django)
